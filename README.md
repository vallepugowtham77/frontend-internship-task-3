# Student Management Dashboard

## 📌 Project Purpose
This project is a frontend dashboard to manage student records dynamically.

## 🚀 Features
- Add student details
- Display student list
- Delete student records
- Form validation
- Responsive design

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## 💡 How to Run
1. Open index.html in browser
2. Add student details
3. Manage records dynamically
